현재 작업을 완료 처리합니다. record-agent를 호출해 재사용 자산을 정리하고 상태를 done으로 변경합니다.

## 실행 절차

1. `.claude/context/current.md` 파일을 읽으세요.

2. 파일이 없으면 "완료할 작업이 없습니다."라고 안내하고 종료하세요.

3. 이미 `status`가 `done`이면 "이미 완료된 작업입니다."라고 안내하고 종료하세요.

4. `record-agent`를 호출해 현재 작업의 재사용 자산을 정리하게 하세요.

5. record-agent 완료 후 아래 형식으로 완료 보고를 출력하세요:

```
✅ 작업 완료 처리 완료

📄 작업 ID: {id}
📝 요구사항: {원본 요구사항 한 줄 요약}

💾 재사용 자산이 .claude/context/current.md에 저장되었습니다.
새 작업을 시작하려면: /task <요구사항>
```
