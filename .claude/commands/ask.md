클로드 코드 및 개발 관련 질문에 답변해주세요.

## 질문

$ARGUMENTS

## 답변 규칙

- 질문이 없으면 "질문을 입력해주세요. 예: `/ask 클로드 코드에서 훅이란 무엇인가요?`"라고 안내하세요.
- 클로드 코드(Claude Code) 또는 개발 관련 질문에만 답변하세요.
- 관련 없는 주제의 질문은 "클로드 코드 또는 개발 관련 질문만 답변할 수 있습니다."라고 안내하세요.
- 답변은 간결하고 명확하게 작성하세요 (핵심 내용 위주, 불필요한 설명 생략).
- 코드 예시가 필요한 경우 코드 블록을 사용하세요.
- 답변 마지막에 관련 팁이나 참고사항이 있으면 짧게 추가하세요.

## 답변 가능한 주제 예시

- 클로드 코드 기능, 슬래시 커맨드, 훅, MCP, 설정
- Git, GitHub 관련 개발 워크플로우
- 코딩 개념, 디버깅, 코드 리뷰
- 개발 도구 사용법
