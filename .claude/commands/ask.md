## 질문

$ARGUMENTS

## 답변 절차

1. 질문이 없으면 "질문을 입력해주세요. 예: `/ask 클로드 코드에서 훅이란 무엇인가요?`"라고 안내하고 종료하세요.
2. 클로드 코드(Claude Code) 또는 개발 관련 질문인지 확인하세요. 관련 없으면 "클로드 코드 또는 개발 관련 질문만 답변할 수 있습니다."라고 안내하고 종료하세요.
3. WebSearch 도구로 질문과 관련된 공식 문서 및 신뢰할 수 있는 사이트를 최소 2곳 검색하세요.
4. 검색 결과를 바탕으로 팩트를 확인한 뒤 답변을 작성하세요.
5. 답변 마지막에 ## 참고 자료 섹션을 추가하고, 실제로 검색해서 확인한 URL을 최소 2개 이상 나열하세요.

## 답변 규칙

- 답변은 간결하고 명확하게 작성하세요 (핵심 내용 위주, 불필요한 설명 생략).
- 코드 예시가 필요한 경우 코드 블록을 사용하세요.
- 답변 마지막에 관련 팁이나 참고사항이 있으면 짧게 추가하세요.
- 참고 자료 URL은 반드시 실제로 WebSearch로 확인한 것만 사용하세요. URL을 추측하거나 임의로 생성하지 마세요.
