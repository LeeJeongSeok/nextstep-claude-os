새 작업을 시작합니다. os-orchestrator를 통해 4단계 워크플로우를 실행합니다.

## 요구사항

$ARGUMENTS

## 실행 절차

1. 요구사항이 없으면 "요구사항을 입력해주세요. 예: `/task 클로드 코드 훅 사용법 정리 문서 작성`"라고 안내하고 종료하세요.

2. `.claude/context/current.md` 파일이 존재하는지 확인하세요.
   - 파일이 있고 `status`가 `done`이 아니면: "진행 중인 작업이 있습니다. `/status`로 확인하거나 `/done`으로 완료 처리 후 새 작업을 시작하세요."라고 안내하고 종료하세요.
   - 파일이 없거나 `status`가 `done`이면: 계속 진행합니다.

3. `os-orchestrator` 에이전트를 호출하세요. 요구사항은 위의 `$ARGUMENTS` 내용입니다.

4. 오케스트레이터가 4단계(해석→구조화→구현→기록)를 모두 완료할 때까지 기다리세요.
